# CAN通信シミュレータ 設計書

## システム概要

Linux SocketCANを用いてCAN通信を模擬するアプリケーションを作成する。

本システムでは送信ノードと受信ノードを実装し、CANフレームの送受信を行う。

学習目的

* C言語による開発
* Linux環境での開発
* SocketCANの利用
* モジュール分割設計
* ヘッダファイルによるインターフェース設計

---

### 実装予定機能
